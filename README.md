## 📋 Sobre o Projeto

Página de destino turístico apresentando **Busan**, a segunda maior cidade da Coreia do Sul. O projeto exibe uma introdução sobre a cidade, seguida de três atrações imperdíveis com imagem, descrição e categorias de público. O foco foi na construção de uma página informativa e visualmente organizada, unindo boas práticas de **HTML semântico** com **estilização consistente em CSS**.

---

## ✨ Funcionalidades

- ✅ Header com headline e título de destaque
- ✅ Seção de descrição com imagem e texto introdutório
- ✅ Lista de 3 cards de atrações turísticas com imagem e descrição
- ✅ Tags de categoria por tipo de público (história, casais, famílias, orçamento)
- ✅ Tipografia clara com fonte **Open Sans** (Google Fonts)
- ✅ Rodapé com link para o autor

## 🧠 Foco Técnico — Estrutura de Landing Page

Este projeto foi pensado para consolidar a construção de páginas completas e bem estruturadas. Alguns conceitos aplicados:

- Uso de `<header>`, `<main>`, `<section>` e `<footer>` para organização semântica
- `<ul>` e `<li>` para estruturar os cards de atrações de forma semântica
- Separação visual de seções com `<hr>` e espaçamentos CSS
- Hierarquia de títulos com `<h1>`, `<h2>` e `<h3>` respeitando a leitura lógica
- `<img>` com `alt` descritivo em todas as imagens para acessibilidade
- Uso de `<strong>` para destacar informações relevantes dentro do conteúdo
